x2 Rounds Argon2id + SHA512 as salt PoW

탈중앙화. 안전. Argon2id.

Dpowcoin Core는 Argon2id 작업증명 알고리즘을 구현한 오픈소스 암호화폐입니다 — ASIC 저항성, 메모리 집약적이며 공정한 채굴을 위해 설계되었습니다.

Argon2id 알고리즘

ASIC 채굴에 저항하는 메모리 집약적 작업증명. CPU 접근성을 통한 공정한 분배.

네트워크 업그레이드

블록 225,000에서 Yespower가 합의에서 제거됩니다 — 이후 Dpowcoin은 Argon2id 전용이 됩니다.

배경: 듀얼 PoW 시대

블록 225,000 이전에는 Dpowcoin이 하나의 타겟을 공유하는 두 개의 독립적인 작업증명 검사로 모든 헤더를 검증했습니다: hash = { Yespower(header), Argon2id(header) }, 유효 ⇔ 두 해시 모두 ≤ target(nBits). Yespower가 먼저 저비용 필터로 검증되었으며, Yespower를 통과한 경우에만 Argon2id가 계산되었습니다.

Yespower를 제거하는 이유

두 알고리즘이 동일한 타겟을 공유했기 때문에, Yespower는 Argon2id가 이미 제공하는 것 이상의 실질적인 보안을 제공하지 못했으며, LWMA가 고려해야 하는 작업량을 두 배로 늘리고 네트워크에 측정 가능한 이점 없이 코드베이스를 복잡하게 만들었습니다. 두 검사가 독립적이므로 Yespower를 제거해도 난이도 조정이나 체인 가중치는 변경되지 않습니다 — 둘 다 알고리즘과 완전히 무관합니다. 듀얼 PoW라는 아이디어는 시도할 가치가 있었지만, 이 구현은 해결한 것보다 더 많은 문제를 만들어냈습니다. “듀얼 PoW”라는 이름은 역사적인 이유로만 유지됩니다.

별도의 조치가 필요하지 않습니다. Yespower와 Argon2id는 항상 동일한 타겟을 공유했기 때문에, Argon2id 기반 채굴 소프트웨어는 블록 225,000 전후로 변경 없이 계속 작동합니다. SHA-512 및 Argon2id 라이브러리는 거의 모든 플랫폼에서 사용할 수 있으므로, 경량 SPV 지갑은 검증을 위해 Argon2id만 사용할 수 있습니다.

중요 고지 사항

당사는 “지연된 PoW” 기술을 사용하지 않으며 Komodo 또는 그 프로젝트와 어떠한 관련도 없습니다. 블록 225,000 이전에 Dpowcoin은 Yespower와 Argon2id로 구성된 듀얼 PoW를 사용했으며, Argon2id 자체는 이중 SHA-512로 솔트 처리된 두 개의 연쇄 라운드로 계산됩니다.
