Завантажити Bitweb Core

Оберіть платформу та перевірте завантаження.

Перевірте завантаження

Завжди перевіряйте контрольну суму SHA256 та підпис GPG перед запуском будь-якого бінарного файлу.

1. Перевірте контрольну суму SHA256

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. Перевірте підпис GPG

Імпортуйте ключ підпису:

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Перевірте SHA256SUMS:

gpg --verify SHA256SUMS.asc SHA256SUMS
Відбиток ключа:
23B3D882F805A5D1F0A12A2535A7538B1C816E49
