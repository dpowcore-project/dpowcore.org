Privacy Policy

Last updated: July 2026

Introduction

By visiting this website, you are accepting the practices described in this policy. This site (dpowcore.org) is a static site: the pages, styles, and scripts in this repository do not themselves collect, store, or process any personal information. Some visitor data is nonetheless handled by the infrastructure this site runs on — described below — which is outside this project's control and governed by that infrastructure's own privacy policy.

Hosting and third-party infrastructure

  • Hosting. This site is hosted on GitHub Pages. Standard access data (such as IP address, request time, and requested page) is logged by GitHub's own infrastructure as part of serving the site, not by any server this project operates. See GitHub's Privacy Statement for what GitHub collects and retains.
  • CDN-delivered assets. A small number of static libraries (Bootstrap, Font Awesome) may be served from a third-party CDN. Requests for those files are visible to the CDN operator in the same way any web request is visible to the server handling it.
  • DDoS / edge protection. If this domain is placed behind a reverse proxy or DDoS-protection service (such as Cloudflare), that service may log requests and set its own cookies as part of distinguishing legitimate traffic from abusive traffic. That layer operates independently of this repository's code.

Cookies

This site's own code does not set any cookies. Any cookie you see while visiting may come from the hosting or edge-protection infrastructure described above. See our Cookie Policy for details, and your browser settings for how to control cookies.

What we use visitor data for

To the extent any visitor data is available to us (for example, aggregate traffic statistics exposed by GitHub Pages), we use it only to understand and improve the availability, performance, and content of this website. We do not run our own analytics or tracking scripts.

Disclosure of information

We do not sell or disclose visitor data to third parties. To the limited extent any data passes through the hosting or CDN infrastructure described above, disclosure is governed by that provider's own policy, not by us — we do not have access to give or withhold beyond what is stated here.

Changes

This Privacy Policy may change from time to time. Any changes will be posted on this page. We will not materially reduce the protections described here for information already collected without prior notice.
