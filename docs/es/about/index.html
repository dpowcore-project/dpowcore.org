Acerca de Bitweb Core

Bitweb Core es un proyecto de código abierto que mantiene y publica el software cliente Bitweb (BTE). Basado en Bitcoin Core, proporciona tanto un nodo completo para validar completamente la blockchain como una cartera integrada. Cualquiera es bienvenido a contribuir.

Esta lista se irá ampliando. Esperamos que Bitweb Core crezca y forme una gran comunidad de desarrolladores de código abierto con muchos colaboradores en el código.

Mantenedores

Los mantenedores del proyecto tienen acceso de confirmación y son responsables de fusionar parches de los colaboradores. Desempeñan un papel de gestión — fusionando parches que el equipo acuerda que deben fusionarse — y actúan como verificación final para garantizar que los parches sean seguros y estén en línea con los objetivos del proyecto. El papel de los mantenedores es por acuerdo de los colaboradores del proyecto.

Colaboradores

Cualquiera puede proponer cambios de código y probar, revisar y comentar las solicitudes de extracción abiertas. Cualquiera que contribuya con código, revisiones, pruebas, traducciones o documentación a Bitweb Core se considera colaborador. Las notas de versión de cada lanzamiento de Bitweb Core contienen una sección de créditos que reconoce a todos los que contribuyeron durante el ciclo de lanzamiento anterior.
