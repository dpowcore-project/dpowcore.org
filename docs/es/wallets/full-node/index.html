Verificar tu descarga

Verifica siempre el checksum SHA256 y la firma GPG antes de ejecutar cualquier binario.

1. Verificar checksum SHA256

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash dpowcoin-*.zip -Algorithm SHA256

2. Verificar firma GPG

Importar la clave de firma:

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Verificar SHA256SUMS:

gpg --verify SHA256SUMS.asc SHA256SUMS
