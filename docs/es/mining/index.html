Minería de Dpowcoin

Dpowcoin utiliza el algoritmo de prueba de trabajo Argon2id — mineable con CPU.

Minería en Solitario

Mina directamente en tu propio nodo completo. Conservas el 100% de las recompensas de bloque.

1. Configurar tu nodo

Añade lo siguiente a dpowcoin.conf:

rpcuser=user
rpcpassword=password
rpcallowip=127.0.0.1
rpcport=42002
txindex=1
coinstatsindex=1
server=1
listen=1

2. Descargar cpuminer-opt

Compilaciones para Windows y Linux — elige el binario que coincida con tu CPU:

bitweb-project/cpuminer-opt

CPU
