Contribuție

Bitweb Core este un proiect cu contribuție deschisă — oricine este binevenit să contribuie prin revizuire de la egal la egal, testare și patch-uri. Nu există o clasă privilegiată de dezvoltatori; contribuitorii câștigă încredere prin merit. Mentenanții repository-ului sunt responsabili de fuzionarea pull request-urilor și de ciclul de lansare.
