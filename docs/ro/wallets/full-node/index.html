Verifică descărcarea

Verifică întotdeauna suma de control SHA256 și semnătura GPG înainte de a rula orice binar.

1. Verifică suma SHA256

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. Verifică semnătura GPG

Importă cheia de semnare:

gpg --keyserver keys.openpgp.org     --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Verifică SHA256SUMS:

gpg --verify SHA256SUMS.asc SHA256SUMS
Amprentă cheie:
23B3D882F805A5D1F0A12A2535A7538B1C816E49
Verifică cheia pe OpenPGP
