Despre Dpowcoin Core

Dpowcoin Core este un proiect open source care menține și lansează software-ul client Dpowcoin (DPC). Bazat pe Bitcoin Core, oferă atât un nod complet pentru validarea completă a blockchain-ului, cât și un portofel integrat. Oricine este binevenit să contribuie.

Mentenanț principal

Această listă va fi extinsă. Sperăm că Dpowcoin Core va crește o mare comunitate de dezvoltatori open source cu mulți contribuitori la cod.

Mentenanți

Mentenanții proiectului au acces la commit și sunt responsabili de fuzionarea patch-urilor de la contribuitori. Aceștia joacă un rol de organizare — fuzionând patch-urile pe care echipa le consideră potrivite — și acționează ca verificare finală pentru a se asigura că patch-urile sunt sigure și în conformitate cu obiectivele proiectului. Rolul mentenanților este stabilit prin acordul contribuitorilor proiectului.

Contribuitori

Oricine este liber să propună modificări de cod și să testeze, revizuiască și comenteze la Pull Request-urile deschise. Oricine contribuie cu cod, revizuiri, teste, traduceri sau documentație la Dpowcoin Core este considerat contribuitor. Notele de lansare ale fiecărei versiuni Dpowcoin Core conțin o secțiune de mulțumiri care recunoaște toți cei care au contribuit în ciclul de lansare anterior.
