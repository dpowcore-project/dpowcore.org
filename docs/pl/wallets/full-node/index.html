Weryfikacja pobrania

Zawsze weryfikuj sumę kontrolną SHA256 i podpis GPG przed uruchomieniem jakiegokolwiek pliku binarnego.

1. Sprawdź sumę kontrolną SHA256

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. Zweryfikuj podpis GPG

Zaimportuj klucz podpisujący:

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Zweryfikuj SHA256SUMS:

gpg --verify SHA256SUMS.asc SHA256SUMS
Odcisk klucza:
23B3D882F805A5D1F0A12A2535A7538B1C816E49
