Współtworzenie

Bitweb Core to otwarty projekt dla współtwórców — każdy może wnosić wkład poprzez wzajemną recenzję, testowanie i łatki. Nie istnieje uprzywilejowana klasa deweloperów; współtwórcy zdobywają zaufanie przez zasługi. Opiekunowie repozytorium odpowiadają za scalanie pull requestów i cykl wydań.
