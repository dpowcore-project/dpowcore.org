Ciclo di Vita delle Versioni

Bitweb Core mantiene aggiornamenti di sicurezza per le due serie di rilascio principale più recenti. Le serie più vecchie raggiungono il Fine Vita (EOL) quando una serie più nuova le sostituisce.
