Contribuzione

Bitweb Core è un progetto aperto ai contributori — chiunque è benvenuto a contribuire tramite revisione tra pari, test e patch. Non esiste una classe privilegiata di sviluppatori; i contributori guadagnano fiducia attraverso il merito. I maintainer del repository sono responsabili dell'unione delle pull request e del ciclo di rilascio.
