Portafoglio Web

Portafoglio basato su browser — nessuna installazione richiesta. Le tue chiavi rimangono sul tuo dispositivo.

Questo è un portafoglio web non custodial. Non condividere mai la tua frase seed o le chiavi private con nessuno.
Servizio Tipo URL
Portafoglio Web Non custodial webwallet2.bitwebcore.net
API Portafoglio REST API api2.bitwebcore.net
