Verifica il tuo download

Verifica sempre il checksum SHA256 e la firma GPG prima di eseguire qualsiasi binario.

1. Verifica il checksum SHA256

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. Verifica la firma GPG

Importa la chiave di firma:

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Verifica SHA256SUMS:

gpg --verify SHA256SUMS.asc SHA256SUMS
Impronta della chiave:
23B3D882F805A5D1F0A12A2535A7538B1C816E49
