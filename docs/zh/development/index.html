贡献

Bitweb Core 是一个开放贡献者项目 — 欢迎任何人通过同行评审、测试和补丁做出贡献。没有特权开发者阶层；贡献者通过功绩赢得信任。仓库维护者负责合并拉取请求和发布周期。
