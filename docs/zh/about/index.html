关于 Dpowcoin Core

Dpowcoin Core 是一个维护和发布 Dpowcoin (DPC) 客户端软件的开源项目。基于 Bitcoin Core，它同时提供用于完全验证区块链的全节点和内置钱包。欢迎任何人参与贡献。

此列表将会扩展。我们希望 Dpowcoin Core 能够发展成为一个拥有众多代码贡献者的大型开源开发者社区。

维护者

项目维护者拥有提交权限，负责合并贡献者的补丁。他们扮演管理角色 — 合并团队同意应该合并的补丁 — 并作为最终检查确保补丁安全且符合项目目标。维护者的角色由项目贡献者协商确定。

贡献者

任何人都可以自由提议代码更改，并测试、审查和评论开放的拉取请求。任何为 Dpowcoin Core 贡献代码、审查、测试、翻译或文档的人都被视为贡献者。每个 Dpowcoin Core 版本的发布说明都包含一个致谢部分，表彰所有在上一发布周期中做出贡献的人。
