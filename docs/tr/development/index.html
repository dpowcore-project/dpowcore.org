Katkı Sağla

Bitweb Core, katkıda bulunanlara açık bir projedir — herkes kod incelemesi, test ve yamalar yoluyla katkıda bulunabilir. Ayrıcalıklı bir geliştirici sınıfı yoktur; katkıda bulunanlar güveni liyakatla kazanır. Depo bakımcıları pull request birleştirme ve sürüm döngüsünden sorumludur.
