О Bitweb Core

Bitweb Core — проект с открытым исходным кодом, поддерживающий клиентское ПО Bitweb (BTE). Основан на Bitcoin Core и включает полную ноду для валидации блокчейна и встроенный кошелёк. Участие открыто для всех.

Команда

Mraksoll
Главный мейнтейнер

Список будет расширяться. Мы надеемся, что вокруг Bitweb Core сформируется большое сообщество разработчиков с открытым исходным кодом.

Мейнтейнеры

Мейнтейнеры проекта имеют доступ к коммитам и отвечают за принятие патчей от контрибьюторов. Они выполняют роль технического фильтра — принимают патчи, с которыми согласна команда, и обеспечивают финальную проверку на безопасность и соответствие целям проекта. Роль мейнтейнеров определяется по соглашению участников.

Контрибьюторы

Каждый может предлагать изменения кода и тестировать, рецензировать и комментировать открытые Pull Request'ы. Любой, кто вносит вклад в код, рецензии, тесты, переводы или документацию Bitweb Core, считается контрибьютором. В заметках к каждому релизу есть раздел credits с упоминанием всех, кто внёс вклад за предыдущий цикл.
