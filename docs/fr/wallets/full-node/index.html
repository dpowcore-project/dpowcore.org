Vérifiez votre téléchargement

Vérifiez toujours la somme de contrôle SHA256 et la signature GPG avant d'exécuter un binaire.

1. Vérifier la somme SHA256

Linux / macOS :

sha256sum --check SHA256SUMS

Windows (PowerShell) :

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. Vérifier la signature GPG

Importer la clé de signature :

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

Vérifier SHA256SUMS :

gpg --verify SHA256SUMS.asc SHA256SUMS
Empreinte de la clé :
23B3D882F805A5D1F0A12A2535A7538B1C816E49
