Minerando Dpowcoin

O Dpowcoin usa o algoritmo de prova de trabalho Argon2id — minerável com CPU.

Mineração Solo

Mine diretamente no seu próprio nó completo. Você fica com 100% das recompensas de bloco.

1. Configure seu nó

Adicione o seguinte ao dpowcoin.conf:

rpcuser=user
rpcpassword=password
rpcallowip=127.0.0.1
rpcport=42002
txindex=1
coinstatsindex=1
server=1
listen=1

2. Baixar cpuminer-opt

Compilações para Windows e Linux — escolha o binário compatível com sua CPU:

bitweb-project/cpuminer-opt

CPU
