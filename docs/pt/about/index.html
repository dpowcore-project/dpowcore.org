Sobre o Bitweb Core

O Bitweb Core é um projeto de código aberto que mantém e lança o software cliente Bitweb (BTE). Baseado no Bitcoin Core, fornece tanto um nó completo para validação total da blockchain quanto uma carteira integrada. Qualquer pessoa é bem-vinda para contribuir.

Esta lista será expandida. Esperamos que o Bitweb Core cresça uma grande comunidade de desenvolvedores de código aberto com muitos contribuidores ao código.

Mantenedores

Os mantenedores do projeto têm acesso de commit e são responsáveis por mesclar patches de contribuidores. Eles desempenham um papel de organização — mesclando patches que a equipe concorda que devem ser mesclados — e atuam como verificação final para garantir que os patches sejam seguros e estejam alinhados com os objetivos do projeto. O papel dos mantenedores é definido por acordo entre os contribuidores do projeto.

Contribuidores

Qualquer pessoa pode propor alterações de código e testar, revisar e comentar em Pull Requests abertos. Quem contribuir com código, revisões, testes, traduções ou documentação para o Bitweb Core é considerado contribuidor. As notas de versão de cada lançamento do Bitweb Core contêm uma seção de créditos reconhecendo todos que contribuíram no ciclo de lançamento anterior.
