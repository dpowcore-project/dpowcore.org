Contribuindo

O Bitweb Core é um projeto de contribuição aberta — qualquer pessoa é bem-vinda para contribuir por meio de revisão por pares, testes e patches. Não existe uma classe privilegiada de desenvolvedores; os contribuidores ganham confiança pelo mérito. Os mantenedores do repositório são responsáveis por mesclar pull requests e pelo ciclo de lançamentos.
