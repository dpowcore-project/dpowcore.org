Argon2id PoW

Descentralizado. Seguro. Argon2id.

O Bitweb Core é uma criptomoeda de código aberto que implementa o algoritmo de prova de trabalho Argon2id — resistente a ASICs, intensivo em memória e construído para mineração justa.

Algoritmo Argon2id

Prova de trabalho intensiva em memória, resistente à mineração por ASICs. Distribuição justa com acessibilidade para CPU e GPU.

Código Aberto

Totalmente de código aberto sob a licença MIT. Auditável, bifurcável e com desenvolvimento orientado pela comunidade no GitHub.

Compilações Reproduzíveis

Compilações determinísticas via GNU Guix. Cada binário é verificável de forma independente. Atestações de lançamento assinadas com GPG.

Relançamento da Rede e Desbloqueio Gradual

O BTE é um relançamento completo da rede com uma gênese limpa, mineração justa e proteção de desbloqueio gradual para moedas distribuídas.

Histórico da Rede

O BTE é o relançamento completo de uma cadeia anterior. Os detentores da rede antiga receberam BTE novos numa proporção de 100:1 — um novo BTE para cada 100 moedas antigas. Para manter a integridade matemática e estética, os primeiros 60.000 blocos foram minerados da maneira padrão: uma recompensa fixa de 50 BTE por bloco, sem saídas coinbase infladas, sem atalhos — mineração pura como base para uma distribuição justa.

Proteção de Desbloqueio Gradual

Para evitar pressão de venda repentina da oferta distribuída, todas as saídas coinbase mineradas durante o período estendido (blocos 60.000–69.300) possuem um bloqueio de maturidade estendido. O bloqueio é projetado para que cada moeda desse período fique disponível para gasto no mesmo momento — bloco 69.400 — dando à rede tempo para encontrar um equilíbrio de preço natural antes que as moedas desbloqueadas possam ser movidas.

Nota para Mineradores

Seus blocos minerados e pagos normalmente — recompensa padrão de 50 BTE em cada altura. Se o seu bloco cair dentro da janela estendida (60.000–69.300), seu coinbase amadurecerá mais tarde que o normal, mas todas essas moedas serão desbloqueadas juntas no bloco 69.400, aproximadamente 32 dias após o início da janela. Fora deste intervalo, aplica-se sempre a maturidade padrão de 100 blocos (~8 horas).

Pools de Mineração — Consolidação de UTXOs

Um pool que ganha 50 BTE por bloco acumulará mais de 1.600 saídas coinbase não gastas ao atingir 80.000 BTE. Criar uma transação de pagamento com tantas entradas pode exceder o limite de tamanho da transação e retornar um erro. Antes de cada execução de pagamento programada, consolide sua carteira: envie lotes de no máximo 50.000 BTE para seu próprio endereço em transações separadas. Execute a consolidação após o bloco 69.400 para evitar erros de maturidade estendida.

Antes de pagamentos grandes: consolide em lotes de ≤ 50.000 BTE
