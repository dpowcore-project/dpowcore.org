Bitweb Coreをダウンロード

プラットフォームを選択してダウンロードを確認してください。

ダウンロードの確認

バイナリを実行する前に、必ずSHA256チェックサムとGPG署名を確認してください。

1. SHA256チェックサムの確認

Linux / macOS:

sha256sum --check SHA256SUMS

Windows (PowerShell):

Get-FileHash bitweb-*.zip -Algorithm SHA256

2. GPG署名の確認

署名キーのインポート:

gpg --keyserver keys.openpgp.org --recv-keys 23B3D882F805A5D1F0A12A2535A7538B1C816E49

SHA256SUMSの確認:

gpg --verify SHA256SUMS.asc SHA256SUMS
キーフィンガープリント:
