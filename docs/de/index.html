Argon2id PoW

Dezentralisiert. Sicher. Argon2id.

Bitweb Core ist eine Open-Source-Kryptowährung mit dem Argon2id Proof-of-Work-Algorithmus — ASIC-resistent, speicherintensiv und für faires Mining entwickelt.

Argon2id-Algorithmus

Speicherintensiver Proof-of-Work, resistent gegen ASIC-Mining. Faire Verteilung durch CPU- und GPU-Zugänglichkeit.

Open Source

Vollständig quelloffen unter der MIT-Lizenz. Überprüfbar, forkbar, gemeinschaftsgetriebene Entwicklung auf GitHub.

Reproduzierbare Builds

Deterministische Builds über GNU Guix. Jedes Binary ist unabhängig überprüfbar. GPG-signierte Release-Attestierungen.

Netzwerk-Neustart & Vesting

BTE ist ein vollständiger Netzwerk-Neustart mit sauberem Genesis, fairem Mining und eingebautem Vesting-Schutz für verteilte Coins.

Netzwerk-Hintergrund

BTE ist ein vollständiger Neustart einer früheren Kette. Inhaber des alten Netzwerks erhielten neues BTE im Verhältnis 100:1 — ein neues BTE pro 100 alte Coins. Um mathematische und ästhetische Integrität zu wahren, wurden die ersten 60.000 Blöcke auf die Standardweise gemint: eine flache Belohnung von 50 BTE pro Block, keine aufgeblähten Coinbase-Ausgaben, keine Abkürzungen — reines Mining als Grundlage für faire Verteilung.

Vesting-Schutz

Um plötzlichen Verkaufsdruck durch die verteilte Versorgung zu verhindern, tragen alle Coinbase-Ausgaben, die während der verlängerten Periode gemint wurden (Blöcke 60.000–69.300), eine erweiterte Reife-Sperre. Die Sperre ist so gestaltet, dass jeder Coin aus dieser Periode zum gleichen Zeitpunkt ausgabefähig wird — Block 69.400 — und dem Netzwerk Zeit gibt, ein natürliches Preisgleichgewicht zu finden, bevor entsperrte Coins bewegt werden können.

Hinweis für Miner

Ihre Blöcke werden normal gemint und ausgezahlt — standardmäßige 50 BTE-Belohnung bei jeder Höhe. Wenn Ihr Block in das erweiterte Fenster fällt (60.000–69.300), reift Ihre Coinbase später als üblich, aber alle solchen Coins werden gemeinsam bei Block 69.400 entsperrt, ungefähr 32 Tage nach Beginn des Fensters. Außerhalb dieses Bereichs gilt wie immer die Standard-100-Block-Reife (~8 Stunden).

Mining-Pools — UTXO-Konsolidierung

Ein Pool, der 50 BTE pro Block verdient, wird bis zum Zeitpunkt, an dem er 80.000 BTE hält, über 1.600 unausgegebene Coinbase-Ausgaben angehäuft haben. Das Erstellen einer Auszahlungstransaktion aus so vielen Eingaben kann das Transaktionsgrößenlimit überschreiten und einen Fehler zurückgeben. Konsolidieren Sie Ihre Wallet vor jedem geplanten Auszahlungslauf: Senden Sie Stapel von maximal 50.000 BTE in separaten Transaktionen an Ihre eigene Adresse. Führen Sie die Konsolidierung nach Block 69.400 durch, um Fehler bei erweiterter Reife zu vermeiden.

Vor großen Auszahlungen: in Stapeln von ≤ 50.000 BTE konsolidieren
