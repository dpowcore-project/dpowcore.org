Web Wallet

Browser-basierte Wallet — keine Installation erforderlich. Ihre Schlüssel bleiben auf Ihrem Gerät.

Dies ist eine nicht-verwaltete Web-Wallet. Teilen Sie niemals Ihre Seed-Phrase oder privaten Schlüssel mit jemandem.
