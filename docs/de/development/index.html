Mitwirken

Bitweb Core ist ein offenes Beitragsprojekt — jeder kann über Peer-Review, Tests und Patches beitragen. Es gibt keine privilegierte Entwicklerklasse; Mitwirkende verdienen Vertrauen durch Verdienst. Repository-Maintainer sind für das Zusammenführen von Pull Requests und den Release-Zyklus verantwortlich.
