Über Bitweb Core

Bitweb Core ist ein Open-Source-Projekt, das die Bitweb (BTE) Client-Software pflegt und veröffentlicht. Basierend auf Bitcoin Core bietet es sowohl einen Full-Node zur vollständigen Blockchain-Validierung als auch eine eingebaute Wallet. Jeder ist eingeladen, beizutragen.

Maintainer

Projekt-Maintainer haben Commit-Zugriff und sind für das Zusammenführen von Patches der Mitwirkenden verantwortlich. Sie übernehmen eine Pflege-Rolle — das Zusammenführen von Patches, über die das Team einig ist — und dienen als letzte Kontrolle, um sicherzustellen, dass Patches sicher und im Einklang mit den Projektzielen sind. Die Rolle der Maintainer erfolgt im Einvernehmen der Projektmitwirkenden.

Mitwirkende

Jeder kann Codeänderungen vorschlagen und offene Pull Requests testen, überprüfen und kommentieren. Jeder, der Code, Reviews, Tests, Übersetzungen oder Dokumentation zu Bitweb Core beiträgt, gilt als Mitwirkender. Die Release-Notizen für jede Bitweb Core-Version enthalten einen Danksagungs-Abschnitt, der alle anerkennt, die im vorangegangenen Release-Zyklus beigetragen haben.
