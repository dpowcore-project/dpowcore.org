--- name: getrpcinfo dpcversion: 26.1.0 dpcgroup: control permalink: en/doc/26.1.0/rpc/control/getrpcinfo/ --- getrpcinfo Returns details of the RPC server. Result: { (json object) "active_commands" : [ (json array) All active commands { (json object) Information about an active command "method" : "str", (string) The name of the RPC command "duration" : n (numeric) The running time in microseconds }, ... ], "logpath" : "str" (string) The complete file path to the debug log } Examples: > dpowcoin-cli getrpcinfo > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getrpcinfo", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:42002/
