--- name: keypoolrefill dpcversion: 26.1.0 dpcgroup: wallet permalink: en/doc/26.1.0/rpc/wallet/keypoolrefill/ --- keypoolrefill ( newsize ) Fills the keypool. Requires wallet passphrase to be set with walletpassphrase call if wallet is encrypted. Arguments: 1. newsize (numeric, optional, default=1000, or as set by -keypool) The new keypool size Result: null (json null) Examples: > dpowcoin-cli keypoolrefill > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "keypoolrefill", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:42002/
