--- name: signmessage dpcversion: 26.1.0 dpcgroup: wallet permalink: en/doc/26.1.0/rpc/wallet/signmessage/ --- signmessage "address" "message" Sign a message with the private key of an address Requires wallet passphrase to be set with walletpassphrase call if wallet is encrypted. Arguments: 1. address (string, required) The dpowcoin address to use for the private key. 2. message (string, required) The message to create a signature of. Result: "str" (string) The signature of the message encoded in base 64 Examples: Unlock the wallet for 30 seconds > dpowcoin-cli walletpassphrase "mypassphrase" 30 Create the signature > dpowcoin-cli signmessage "P8pqp4xfWKN6s239iTLBLKrYtnjpURRf8u" "my message" Verify the signature > dpowcoin-cli verifymessage "P8pqp4xfWKN6s239iTLBLKrYtnjpURRf8u" "signature" "my message" As a JSON-RPC call > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "signmessage", "params": ["P8pqp4xfWKN6s239iTLBLKrYtnjpURRf8u", "my message"]}' -H 'content-type: text/plain;' http://127.0.0.1:42002/
