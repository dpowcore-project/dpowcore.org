--- name: dumpprivkey dpcversion: 26.1.0 dpcgroup: wallet permalink: en/doc/26.1.0/rpc/wallet/dumpprivkey/ --- dumpprivkey "address" Reveals the private key corresponding to 'address'. Then the importprivkey can be used with this output Note: This command is only compatible with legacy wallets. Arguments: 1. address (string, required) The dpowcoin address for the private key Result: "str" (string) The private key Examples: > dpowcoin-cli dumpprivkey "myaddress" > dpowcoin-cli importprivkey "mykey" > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "dumpprivkey", "params": ["myaddress"]}' -H 'content-type: text/plain;' http://127.0.0.1:42002/
