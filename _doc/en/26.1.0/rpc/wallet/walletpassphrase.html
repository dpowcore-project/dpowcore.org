--- name: walletpassphrase dpcversion: 26.1.0 dpcgroup: wallet permalink: en/doc/26.1.0/rpc/wallet/walletpassphrase/ --- walletpassphrase "passphrase" timeout Stores the wallet decryption key in memory for 'timeout' seconds. This is needed prior to performing transactions related to private keys such as sending dpowcoins Note: Issuing the walletpassphrase command while the wallet is already unlocked will set a new unlock time that overrides the old one. Arguments: 1. passphrase (string, required) The wallet passphrase 2. timeout (numeric, required) The time to keep the decryption key in seconds; capped at 100000000 (~3 years). Result: null (json null) Examples: Unlock the wallet for 60 seconds > dpowcoin-cli walletpassphrase "my pass phrase" 60 Lock the wallet again (before 60 seconds) > dpowcoin-cli walletlock As a JSON-RPC call > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "walletpassphrase", "params": ["my pass phrase", 60]}' -H 'content-type: text/plain;' http://127.0.0.1:42002/
